Lisp-ed is a project to make a lisp editor, with a lot of tools on it. There is still a lot to be done.(The TODOs are not exhaustive.) I also hope to share/combine work with Able.

Lisp-ed

Currently able to use both regular and emacs-like copy-pasting and such. The function that starts the editor has an argument that is a function that makes various bindings. The definition of this function, however reads like a list. This way, one can just compose functions to set ones the preferences.

Of course, code can be executed right from the buffer.

Autodoc

Automatic documentation, that uses LML2 and documentation from code-scan. Uses mk-website, and can do both all-on one page, and separate pages.
