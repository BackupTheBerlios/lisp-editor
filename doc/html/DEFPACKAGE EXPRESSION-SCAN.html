Package: Expression-scan

Can create and use expression-hook to obtain information about code. Any s-expression can be tracked. (So macros and functions can be tracked.)

Uses packages: Cl, Generic, Package-stuff, Expression-hook

variable *scan-result*

Lists result by name.

variable *additional-scan*

List of functions for scanning besides every expression. (Rather then fun-scan, elements of which only scan specific macros.

defun scan-loadfile(stream )

Scans a file with a bunch of loads in it.

defun scan-file(stream )

Scans a file as source code in order to document it.

defun scan-expression-hook(expr )

The expression hook of the scanner.

defun scan-expr(expr )

Scans a single expression.

defun fun-scanner(expr )

Function to scan function/macro.

defun add-scanner-fun(for-name scan-function )

Adds a scanner for a macro/function.

defun access-result(fun-name name )

Accesses result of scan of macro/function. Providing a list of fun-names will search them in sequence.

defmacro def-scanner(for-name (args )&body body )

Add-scanner-fun, but does the creation of function for you. Note that the variable expr contains the whole expression.

defgeneric name

Gets name of object.
